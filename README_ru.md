# Трекер коллекций Lineage 2M

> **Внимание:** База данных ещё далека от наполнения. На данный момент добавлены все классы от Уникальных до Редких, все коллекции для классов Уникального и Мифического уровней, а также для Легендарных классов Sedrick и Priscilla. Работа по агатионам ещё только предстоит.

Легкое локальное веб-приложение, созданное для помощи игрокам Lineage 2M в отслеживании их Классов, Агатионов и Коллекций. Создано на Python (FastAPI) и чистом HTML/JS/CSS, имеет быстрый и отзывчивый интерфейс с дизайном в стиле темного фэнтези.

## Основные возможности

- **Управление данными:** Добавляйте Классы и Агатионы, указывая их редкость и доступные пути улучшения (Преобразить, Продвижение, Совмещение, Наделить силой духа).
- **Отслеживание коллекций:** Создавайте коллекции с определенными бонусами статов и требованиями к предметам (включая минимально необходимые уровни улучшений).
- **Личный инвентарь (Моё):** Отмечайте предметы, которые у вас уже есть, и настраивайте их уровни улучшений.
- **Умный поиск (Finder):** Ищите один или сразу несколько нужных статов (например, `Снижение урона` или `Шанс оглушения`). Приложение покажет, какие коллекции дают эти статы, открыты ли они у вас, и **каких именно предметов или улучшений вам не хватает** для их завершения. Также в Finder встроена система рекомендаций, которая анализирует ваш инвентарь и подсказывает **"Топ-10 улучшений"** и **"Самые лёгкие коллекции"** для быстрого закрытия.
- **Импорт / Экспорт:** Легко делайте бэкап всей вашей базы данных (классы, агатионы, коллекции и инвентарь) в JSON файл и восстанавливайте её в любое время. Вы также можете экспортировать/импортировать только ваш личный инвентарь, чтобы без проблем обновлять вашу локальную базу из чужого файла!

## Технологический стек

- **Backend:** Python, FastAPI, Pydantic
- **Database:** SQLite (Локальный файл `app.db` в режиме WAL для производительности)
- **Frontend:** Чистый HTML5, JavaScript (Fetch API) и CSS3 (Свой темный дизайн в духе L2M)

## Требования

- Установленный Python 3.8 или новее.

## Установка и запуск

1. **Откройте терминал** и перейдите в папку проекта:
   ```bash
   cd path/to/L2MColls
   ```

2. **Установите необходимые Python-пакеты:**
   ```bash
   pip install fastapi uvicorn pydantic
   ```

3. **Запустите локальный сервер:**
   Вы можете просто запустить файл `run.bat` двойным кликом (для Windows) или использовать команду в терминале:
   ```bash
   python -m uvicorn main:app --reload
   ```
   *(Флаг `--reload` позволяет серверу автоматически перезапускаться при изменении Python-кода).*

4. **Доступ к приложению:**
   Откройте веб-браузер и перейдите по адресу:
   http://localhost:8000

5. **Документация API (Опционально):**
   FastAPI автоматически генерирует интерактивную документацию. Вы можете посмотреть её здесь:
   http://localhost:8000/docs

## Рекомендуемый порядок работы

**Для обычных пользователей:**
1. **Загрузка базы:** Если кто-то поделился с вами готовой базой, перейдите на страницу **Данные** и нажмите **Импортировать JSON**. Обычно вам не нужно ничего редактировать вручную на странице "Данные", если вы не хотите добавить или исправить что-то сами!
2. **Заполнение инвентаря:** Перейдите на страницу **Моё** и отметьте галочками предметы, которые есть у вашего персонажа, выставив их уровни улучшений.
3. **Поиск улучшений:** Используйте страницу **Finder**, чтобы найти стат, который хотите усилить. Приложение покажет вам, какие коллекции его дают и каких именно предметов не хватает.

**Как обновить локальную базу без потери инвентаря:**
1. Перейдите в **Моё** и нажмите **Экспортировать инвентарь (JSON)**, чтобы безопасно сохранить свой личный прогресс.
2. Перейдите на страницу **Данные** и нажмите **Импортировать JSON**, чтобы загрузить новый, обновленный файл базы данных, который вы получили.
3. Наконец, нажмите **Импортировать инвентарь (JSON)** и выберите ваш ранее сохраненный файл инвентаря, чтобы мгновенно восстановить все ваши отмеченные предметы и уровни улучшений!

**Для тех, кто ведет базу:**
- Используйте страницу **Данные** для ручного заполнения базы новыми Классами, Агатионами и Коллекциями. Как закончите, используйте **Экспортировать JSON**, чтобы поделиться своим трудом с другими.
